## 📖 Overview
An embedded systems capstone project that detects vehicle impact in real time using hardware sensors and automatically alerts emergency contacts via SMS through a GSM module — designed to reduce emergency response time and improve passenger safety.

## 🎯 Objective
To engineer an automated impact-detection architecture that:
- Continuously monitors vehicle sensor data
- Detects collision events in real time
- Sends immediate SMS alerts with location/status data to emergency handlers
- Minimizes cellular communication latency

## 🛠️ Tech Stack
`Embedded C` `Microcontrollers` `GSM Module` `Hardware Sensors (Accelerometer/Vibration)` `IoT Basics`

## ⚙️ How It Works
1. Hardware sensors continuously capture impact/vibration data from the vehicle.
2. The microcontroller processes sensor readings against a threshold to detect a potential accident.
3. On detection, the system triggers the GSM module to send an SMS notification.
4. Real-time data metrics are broadcast to emergency handlers to enable a faster response.
